Mastodon API tester

Let's build glue for scripting Mastodon!

I've started by implementing four verbs.

  1. mastodon.getServerInfo
  2. mastodon.getPublicTimeline
  3. mastodon.getPublicStatusesWithTag
  4. mastodon.getUserStatuses

The test functions are (meant to be called from the console):

  1. testGetServerInfo ()
  2. testGetPublicTimeline ()
  3. testGetPublicStatusesWithTag ()
  4. testGetUserStatuses ()

How to test:

  1. Open the JavaScript console.
  2. Type testGetServerInfo () and press Return. Or one of the other verbs listed above.
  3. The JavaScript object that is returned is displayed in the console.
